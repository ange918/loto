# Loto

## Overview
A React + Vite frontend application.

## Project Architecture
- **Framework**: React 19 with Vite 7
- **Language**: JavaScript (JSX)
- **Entry Point**: `src/main.jsx`
- **Build Output**: `dist/`

## Development
- **Dev Server**: `npm run dev` (port 5000)
- **Build**: `npm run build`
